깃허브 블로그 만들기(hexo 이용)

Git hub blog - hexo

1. 필수 파일 설치

  • nodejs.org 다운로드

    • 설치 시 주의 사항

  • git bash에서 확인

  • hexo 설치

2. 블로그 생성

  • 적당한 폴더 생성(차례대로 실행)
  • 블로그 파일명 생성(차례대로 실행)

  • <자신의 닉네임>.github.io 생성

  • npm 설치

  • _config.yml 파일 설정

    • 사이트 정보 수정

    • 블로그 URL 정보 설정

    • 깃허브 연동

3. 깃허브에 배포

  • 차례대로 실행 후 http://localhost:4000/ 에서 접속 확인
  • 최종적으로 배포
  • 배포 완료 후 SeungKyu37.github.io에 접속해 확인
