R 설치하기

R

  • R : https://www.r-project.org/

    4.1.3버전으로 다운로드

  • Rstudio : https://posit.co/download/rstudio-desktop/

  • Rtools : https://cran.r-project.org/bin/windows/Rtools/rtools40.html

차례대로 설치 후 RStudio 실행

“+” 버튼 클릭 후 R Script 클릭

  • write('PATH="${RTOOLS40_HOME}\\usr\\bin;${PATH}"', file = "~/.Renviron", append = TRUE) 입력 후 실행

  • console에 install.packages("jsonlite", type = "source") 입력 후 실행

  • install.packages(”tidymodels”) 입력 후 실행

  • Tools > global option > code

    • Soft-wrap R source file 체크

      • saving 에서 UTF-8로 바꾸기

자바 설치 방법

Git & Github 설정

다운로드

  • Git : https://git-scm.com/downloads

visual studio code 설치 후

Repositories에서 프로젝트 생성

코드 링크 복사 후

바탕 화면 마우스 우 클릭 Git Bash Here 클릭

git clone Shift+”insert” 입력

바탕 화면에 생성된 프로젝트 우 클릭 후 Git Bash Here 클릭

code . 입력

.gitignore에 gitignore.io에서 가져온 내용 뒷 부분에 복사

New Terminal 클릭

git bash 추가

TERMINAL에

$ git commit -m “updated” 입력

에러 확인 후

git config –global user.email “xmcmrb@naver.com“ 입력

git config –global user.name “SeungKyu37” 입력

git push 입력 후 엔터(defalut)

사이트(Authorize application) 뜨면 확인 누르고 연동 확인

git add . 입력

git commit -*m “your_*msg” 입력

git push 입력

으로 github에 프로젝트 갱신
