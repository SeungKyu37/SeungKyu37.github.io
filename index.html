Git hub blog - hexo

1. 필수 파일 설치

  • nodejs.org 다운로드

    • 설치 시 주의 사항

  • git bash에서 확인

  • hexo 설치

2. 블로그 생성

  • 적당한 폴더 생성(차례대로 실행)
  • 블로그 파일명 생성(차례대로 실행)

  • <자신의 닉네임>.github.io 생성

  • npm 설치

  • _config.yml 파일 설정

    • 사이트 정보 수정

    • 블로그 URL 정보 설정

    • 깃허브 연동

3. 깃허브에 배포

  • 차례대로 실행 후 http://localhost:4000/ 에서 접속 확인
  • 최종적으로 배포
  • 배포 완료 후 SeungKyu37.github.io에 접속해 확인

R 설치하기

R

  • R : https://www.r-project.org/

    4.1.3버전으로 다운로드

  • Rstudio : https://posit.co/download/rstudio-desktop/

  • Rtools : https://cran.r-project.org/bin/windows/Rtools/rtools40.html

차례대로 설치 후 RStudio 실행

“+” 버튼 클릭 후 R Script 클릭

  • write('PATH="${RTOOLS40_HOME}\\usr\\bin;${PATH}"', file = "~/.Renviron", append = TRUE) 입력 후 실행

  • console에 install.packages("jsonlite", type = "source") 입력 후 실행

  • install.packages(”tidymodels”) 입력 후 실행

  • Tools > global option > code

    • Soft-wrap R source file 체크

      • saving 에서 UTF-8로 바꾸기
